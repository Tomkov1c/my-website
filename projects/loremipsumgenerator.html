Lapschool

Generate Lorem Ipsum paragraphs / words / sentences from Flow Launcher.

Simply type lorem to start. Select what you want to generate. Add a number to specify the amount of ___ generated (optional). Press ctrl + v to paste.

Features

  • Generate Words (custom amount)
  • Generate Paragraphs (custom amount)
  • Generate a Sentence

Planed Features

  • Change default sentence length
  • Change default paragraphs length

Installation

Type this in the flow launcher and press enter.

pm install lorem
